# Project: Todo Application

## Project Overview
Build a fully functional todo application with local storage persistence, filtering, and advanced features.

## Features to Implement
1. ✅ Add new todos
2. ✅ Mark todos as complete/incomplete
3. ✅ Edit existing todos
4. ✅ Delete todos
5. ✅ Filter todos (All, Active, Completed)
6. ✅ Clear completed todos
7. ✅ Persist data in localStorage
8. ✅ Todo counter
9. ✅ Drag and drop reordering
10. ✅ Due dates and priorities

## Technical Requirements
- Vanilla JavaScript (no frameworks)
- Responsive design
- Local storage for persistence
- Clean, modular code structure
- Error handling
- Accessibility features
